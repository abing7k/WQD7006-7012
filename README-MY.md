# Arkib Kertas Peperiksaan WQD 7006 / WQD 7012

[English](README.md) | [中文](README-ZH.md)

Repositori ini ialah pusat maklumat dan arkib pembelajaran tidak rasmi untuk kursus **WQD 7006 / WQD 7012: Machine Learning for Data Science / Machine Learning Application** bagi program **Master of Data Science di Universiti Malaya (UM)**.

WQD 7006 dan WQD 7012 merujuk kepada penawaran kursus yang sangat berkaitan dalam bidang pembelajaran yang sama. Bahan di sini dikumpulkan untuk membantu pelajar memahami format peperiksaan terdahulu, gaya soalan, dan fokus ulang kaji.

## Kertas Yang Tersedia

Fail semasa:

- `WQD 7006-final-2018.pdf`
- `WQD 7006-final-2019.pdf`
- `WQD 7006-final-2020.pdf`
- `WQD 7006-final-2024.pdf`
- `WQD 7006-final-2025-Jan.pdf`
- `WQD 7006-final-2025-June.pdf`

## Projek Komuniti

Ini ialah projek perkongsian ilmu pelajar yang bersifat bukan keuntungan.

Semua fail mesti dikongsi **secara percuma**. Jangan menjual, membungkus semula untuk jualan, menjual semula, atau menggunakan kandungan repositori ini untuk akses berbayar, pakej tuisyen, jualan fail persendirian, atau apa-apa tujuan komersial.

Jika anda mempunyai kertas peperiksaan terdahulu, contoh soalan, pembetulan, maklumat metadata, atau cadangan penamaan fail yang lebih tepat, sumbangan melalui pull request amat dialu-alukan.

## Panduan Sumbangan

Apabila menghantar pull request:

- Gunakan nama fail yang jelas seperti `WQD 7006-final-2024.pdf` atau `WQD 7006-middle-2024.pdf`.
- Gunakan `final` untuk kertas peperiksaan akhir.
- Gunakan `middle` untuk peperiksaan pertengahan semester atau midterm.
- Gunakan tahun yang dipaparkan pada halaman pertama PDF.
- Jika terdapat lebih daripada satu kertas pada tahun yang sama, tambah label bulan atau bahagian seperti `Jan`, `June`, atau `Part-2`.

## Penafian

Repositori ini bukan repositori rasmi Universiti Malaya. Nama kursus, kandungan kertas peperiksaan, dan bahan akademik berkaitan adalah milik pemilik, pensyarah, jabatan, dan/atau Universiti Malaya yang berkenaan.

Repositori ini dikekalkan hanya untuk tujuan pembelajaran, ulang kaji, dan rujukan arkib. Jika terdapat bahan yang tidak sepatutnya tersedia secara umum, sila buka issue atau pull request supaya perkara tersebut boleh disemak.
