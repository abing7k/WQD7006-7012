## Community Project

This is a non-profit student knowledge-sharing project.

These files must be shared **free of charge**. Do not sell, package, resell, or use this repository for paid access, tutoring bundles, private file sales, or any other commercial purpose.

If you have additional past papers, sample papers, corrections, metadata, or better file naming information, contributions are welcome through pull requests.

## Contribution Guide

When submitting a pull request:

- Use a clear filename such as `WQD 7006-final-2024.pdf` or `WQD 7006-middle-2024.pdf`.
- Use `final` for final examination papers.
- Use `middle` for midterm or middle-term examination papers.
- Include the year shown on the first page of the PDF.
- If there are multiple papers in the same year, add a short month or part label, such as `Jan`, `June`, or `Part-2`.
